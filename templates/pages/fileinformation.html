How to Create an Information File

The following instructions are for the appropriate assembly of data in a comma-separated values file, or CSV file, in order to receive accurate student STAAR prediction scores.

Simply put, the machine learning models receive data from the information file and after analysis and computation append new computed predictions to the end of the file.

Therefore, in order for the machine learning models to provide the most accurate predictions it is necessary to provide all requested information in the correct order and format.

It is recommended that the following template information file CSV be used for easier assembly of the information file, and to ensure inputted information is accurate.

Download Template Information File CSV

Step 1

As seen in the column headers ‘A’ to ‘F’ of the information file CSV template, the following student data is required for each student:

  • Local Id
    • No space characters can be utilized in this column
  • Grade
    • Please convey by the grade number rather name
  • Ethnicity
    • Use the following abbreviations to convey students’ ethnicities
      • ‘H’ for Hispanic and or Latin Origin
      • ‘W’ for White and or European
      • ‘B’ for African American
  • ECD (Early Childhood Development)
    • Please use ‘1’s and ’0’s for ‘yes’ and ‘no’
  • LEP (Limited English Proficiency)
    • Please use ‘1’s and ’0’s for ‘yes’ and ‘no’
  • SpEd (Special Education)
    • Please use ‘1’s and ’0’s for ‘yes’ and ‘no’

Note, each student should be in a unique row with each student’s information parameter in a unique column in said row. Details on how to convey each information parameter can be seen in the table above.

Step 2

As can be seen in the template column headers ‘G’ through ‘L’ should be populated with student STAAR test scores. These test scores columns can be populated with prior STAAR scores for a student and or benchmark test given by the school district. At this time, it is necessary that each uploaded information file contain scores ONLY from the same core subject area and grade cohort. The STAAR test score data itself should be conveyed as the percent number out of one hundred, excluding any percent signs or decimals. The following are the column header names within the information file CSV template:

  • Fall 2017 Math BM MA.04.005.E PcntScore 12/13/2017
    • Conveyed as the percent number out of one hundred, excluding any percent signs or decimals
  • Spring 2018 Released STAAR BM 3/29/2018
    • Conveyed as the percent number out of one hundred, excluding any percent signs or decimals
  • Spring 2018 STAAR MA04 PcntScore 5/2018
    • Conveyed as the percent number out of one hundred, excluding any percent signs or decimals
  • Fall 2018 Math BM MA.05.018.E PcntScore 12/21/2018
    • Conveyed as the percent number out of one hundred, excluding any percent signs or decimals
  • Spring 2019 Released STAAR BM 2/11/2019
    • Conveyed as the percent number out of one hundred, excluding any percent signs or decimals
  • Spring 2019 STAAR MA05 PcntScore 5/2019 or 6/2019
    • Conveyed as the percent number out of one hundred, excluding any percent signs or decimals

Step 3

Finally, save the populated file as a csv. Note, any file name can be used.

Upon the completion of step three an information file has now been created that will result in accurate student STAAR score predictions. To receive these predictions please upload the file on the District School Performance Predictor File Upload Page using an email-verified account.
